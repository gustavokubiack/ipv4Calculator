# ipv4Calculator

### Prerequisites

* [Node.js](https://nodejs.org/en/)

### Cloning the repository and running the calculator

```
git clone https://github.com/gustavokubiack/ipv4Calculator.git 
```

```
cd ipv4Calculator
```

```
npm i
```

```
npm start
```

### Example

#### Enter IP: 
``` 
192.168.10.0 
``` 
#### Enter Mask: 
``` 
27 
```
